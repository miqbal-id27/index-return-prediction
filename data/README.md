# Data Folder

Place the raw CSV files in `data/raw/` before running the reusable pipeline.

Raw data is intentionally excluded from Git by `.gitignore` to keep the portfolio repository lightweight and avoid publishing private/course datasets.

Expected files:

```text
company_info.csv
index.csv
stock_data.csv
training_targets.csv
testing_targets.csv
fed funds rate.csv                  # optional
fed inflation rate.csv              # optional
fed unemployment rate.csv           # optional
us 5 year treasury.csv              # optional
us 10 year treasury.csv             # optional
vix index.csv                       # optional
```
